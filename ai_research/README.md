# 🧠 EcoVerse AI Research & Machine Learning

This directory serves as the foundation for future AI/ML integrations in EcoVerse. 
Our goal is to eventually replace external API dependencies with in-house classification models for waste analysis and carbon footprint predictions.

## Current Utilities
- `waste_classification_eda.py`: An Exploratory Data Analysis (EDA) baseline script that generates synthetic waste data, analyzes distributions, and exports visualizations.

## Getting Started
To run the research scripts locally, install the required Python data science packages:

```bash
pip install -r requirements.txt
python waste_classification_eda.py
